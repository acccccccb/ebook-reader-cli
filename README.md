# ebook-reader-cli

> 在命令行中读取 epub 文件

## 使用方法

### 安装/调试

```bash
npm i -g https://github.com/acccccccb/ebook-reader-cli.git
cd ebook-reader-cli
npm link
```

### 打开 epub 文件

```bash
erc <epub_file_path>
```

### 阅读快捷键

-   翻页：`PageUp`/`PageDown` `z`/`x`
-   上一章/下一章：`,`/`.`
-   返回目录：`b`
-   回到章节开始：`home`
-   回到章节末尾：`end`
-   关闭书籍并删除阅读记录：`delete`
-   退出：`q` / `ctrl+c`
